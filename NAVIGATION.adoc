== Common Tasks

=== Running the Playground

[source,bash]
----
cd .build/
just run             # Run playground
just dev             # Development mode with watch
----

=== Building

[source,bash]
----
cd .build/
just build           # Build all
just build-rescript  # Build ReScript only
----

=== Testing

[source,bash]
----
cd .build/
just test            # Run all tests
deno test            # Run Deno tests directly
----

== Document Conventions

* **`.adoc`** - AsciiDoc format (preferred for complex docs)
* **`.md`** - Markdown format (for GitHub compatibility)
* **`.scm`** - Guile Scheme (machine-readable metadata)
* **`.res`** - ReScript source code
